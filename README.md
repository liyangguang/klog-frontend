# klog

A vue web app + a basic node server.

## Project setup
```
npm install
```

### Local dev
```bash
npm run dev-fe  # Auto reload vue code
npm run dev-be  # Auto reload node server
```

### Compile and prod

```bash
npm run prod  # compile and run server using forever.js
```

```bash
npm run stop  # stop forever.js
```
